# Relationship Records

This folder captures explicit links between records so the AI and backend do not need to infer everything from prose.

Examples:

- an item is crafted from components
- a part drops from a mission rotation
- a mod is compatible with a class of weapons
- a status effect synergizes with a damage strategy
- a resource is associated with a location

When in doubt, prefer a small relationship record over duplicating the same fact across multiple files.
